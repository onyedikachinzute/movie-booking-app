# Panaview Movie Booking App

An online movie ticket booking application built with Python, PostgreSQL, and Tkinter.  
Users can register, login, view movies and showtimes, select seats, and book tickets.  
Admins can manage movies, shows, and view bookings.

## Prerequisites

- Python 3.x
- PostgreSQL installed and running

## Setup

1. Run the initdb.py file in the main directory (For first time setup only!!!)
2. Run the main.py file in the main directory

Install required Python packages  
`pip install psycopg2`

- Register a new user or login with an existing account.
- If admin, manage movies and shows.
- If user, browse available shows and select seats to book tickets.
